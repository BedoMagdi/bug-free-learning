Common HTML Tags

Tag Name Description
<html> HTML Element All content/structure for an HTML document should be contained between the opening and closing <html> tags.
<h1>-<h6> Heading Elements HTML can use six different levels of heading elements.
<p> Paragraph Element The paragraph element contains and displays a block of text.
<div> Div Element The <div> element is used as a container that divides an HTML document into sections.
<ol> Ordered List Element The <ol> ordered list element creates a list of items in sequential order.
<li> List Item Element The <li> list item element creates a list item.
